平凡的小摸鱼切图仔的2020

2020-12-06
总结

目录

  • 问候
  • 言归正传

看见掘金这次活动头图不错，然后嗖一下，很快啊，现在就是我的了，什么掘金的，在我手机上就是我的。

前言

问候

刚吃完饭，然后xdm吃了吗？没吃的话，来我家吃。

言归正传

一时竟然不知道如何开篇写我得2020年的总结，写之前想的一些骚话，想贴的一堆弔图，不知如何呈现出来，算了，那我年轻人也就不讲章法了，上来就是小学生写作文常用技巧之记流水账 ，但是这个 流水账 也要记得我满意为止，前两天写的一篇文章也花了我很长的时间，但是写出来我并不满意，所以这篇文章不管花多少时间，我一定得写得让我自己满意，一年也就写一次，一年也只有一次让自己满意得机会。要说我的2020年有关键字的话，那就是：平平凡凡，普普通通 ，别无其它。测试。

1.当下

当前写下这句话的时间是：2020年12月5号的晚上20:00，看了下手机正好是个整点。今年这一年说真的实在是过的太快了，呲溜一下就年底了，人不讲武德也就算了，时光也不讲武德，嗖一下，很快啊，我马上就要回家过年，真是感叹时光易逝啊。今天原本是规划学一下红宝书昨晚学剩得小半章内容和第一章以及写完这篇2020年的总结，那小半章倒是学完了，第一章今天估计学不了了，都怪早上起不来，所以今天就只能完成两项了。

2.工作

今年还是待在原来的公司，对自己今年在工作上的评价，中规中矩吧。安排给我的任务我都能完成，毕竟也在公司待那么长时间了，并没有像有些大佬那样能在公司里 叱诧风云 ，我今年在公司里承担的角色仍然是一个小小的前端切图仔。今年也并没有怎么升职加薪，加薪也就加了那么一丢丢，好在我心态还是比较好的，看事情还也很乐观，所以在公司里待的也还算比较舒服，并没有那种上班如上坟的感觉，除了工资不高，现在的公司应该找不出什么大缺点了(后续补充：还有就是一直做业务，提升不大)，虽然身处北京，但是也没见过什么世面(指其它公司的氛围)，因为我从毕业实习以来就一直 呆在现在的公司，中间虽然也想过跳槽，但是因为各种原因都搁置下来了，到现在也有三年多了。然而，现在我又有点蠢蠢欲动了，最近正在学习，看年后能不能挪挪窝，因为实在是

所以年后还是先拿了年终奖，然后 金三银四 去见见世面，年轻人嘛，多见见世面总不是什么坏事，出去看看自己有几斤几两，到底值什么价，如果低于自己的预期就继续努力学，现在正在努力学习之中，只是进度有点慢，但是学得还是蛮细的。

今年开年因为疫情的缘故在家办公，还是比较闲的，然后公司组织一些远程培训，学学公司相关的一些底层的东西，然后后面来公司上班了，好像已经四月份还是五月份了，就去某个银行出差给他们现场写一个系统相关的功能了，虽然我是一个 前端切图仔，但是这差不多三个月的时间写的都是 go 和 node，写一些接口和数据处理相关的代码，当然 go 和 node 也是之前临时学的，现学现卖，因为这项目研发就之前还有一个前端小姐姐还有一个接底层写 C 的，后面写 C 的离职的，然后前端的功能之前的那个小姐姐写的差不多了，然后我接手就是做一些数据处理，然后写一些接口把数据接上的工作，再就是前端有些修改的地方，我这边再修改一下，刚开始前端小姐姐还在的时候，有点嫌弃我写的接口，确实我也是写的不是很好，毕竟我也不是 专业后端 出身，虽然我以前培训过，学过 java ，但是长时间不用我也就差不多忘干净了，后面干脆我自己一个人弄得了。差不多一两个月的时间，每天都是去银行办公，确实东西还是蛮多的。

后面这个活干完就回公司接手一个资源网站的任务，虽然这个架子不是我搭的，但是维护和加功能就需要我来做了，然后是3份代码，一份服务端的 node 写的，一份用户端的，一份管理端的。由于公司人手不够，那时候也正在招人，所以也是我一个人来弄，确实有点弄不过来，但是勉勉强强能让网站正常运行，公司内部的实施和给外面客户展示是没什么问题，但是前面 Bug 还没改完，后面就要加新功能，说 Bug可以先放放，先加新功能，真的是 烂泥糊烂墙，烂上加烂，当时那段时间我的心情真的是用PS把100张表情包拼到一起也不能表达。

但是由于 家境贫寒 ，地主家也没有余粮啊，我还是坚持下来了，资源网站这个做了差不多有一个多月，招了两个新人做，我就撤出来了，刚开始我给他们讲讲代码逻辑，然后他们两个就专门弄这个网站，这个时候我的心情就豁然开朗了，人间值得 ，也有表情包能描述我得心情了 ，至于新来的两个 好兄弟 ，肯定也会觉得人间值得的，两个人正好可以一边做网站，一边🤺🤺，岂不美哉！

我：

然后他们两就一直做网站做到现在，但是我没有跟他们呢坐在一起，要不然那些我遗留给他们的宝贵 Bug，就真是上一秒好兄弟，下一秒兄弟决裂的导火索了，后面我就做项目上的一些定制化的图表啊还有一些其它业务的定制化开发，不算太忙，但是有时候也会加班到很晚，十一二点的样子，不过这种时候不是很多，也能接受，一直做到现在，因为各种不同的项目，就有很多不同的这种需求。最近感觉做这个有点太机械化了，当然可能做项目型的任务的时候都是这样，千篇一律，做的东西都差不多，然后想去搞底层的 API 相关的东西或者去搞 3D 相关的东西，当然现在做的这些东西也还行，也能学到一些东西，就是进步不大，但是这种活总得有人来干啊。然后今年的工作基本就是这样，后几个月可能还是干这个。

3.生活

今年由于疫情，前几个月一直待在家里，生活中也并没有出现 故事会有逻辑，但是生活没有。 这种好事，还是那么的平平淡淡。

翻看我去年写在有道云笔记里的2019年年终总结，发现今年与去年并没有什么大的不同，同样的平淡无味，同样的普普通通，但是，今年生活上还是比去年多那么一丢丢色彩的。

2019年年终总结

首先，今年并没有去年那种半夜一个人去医院挂急诊的情况，去年因为急性胃肠炎和扁桃体化脓两次半夜一个人打车去医院，真的是回忆起来只有两个字，泪目。但是，今年并没有，今年还是挺健康的，没有怎么去医院，除了之前去看了看脸上的痘痘和牙齿之外，基本没进过医院，这点相较于去年真的是好太多了，也不得不夸一句我得身体，今年你真的是太棒了，明年请继续保持。

然后就是今年年初的时候由于没什么事，在家里刷B站，感觉鬼畜挺有意思的，就学着做鬼畜视频，鬼畜调音的那种，然后每天看教学视频，看大佬直播是怎么做鬼畜的啊，后面慢慢也学会了，也做出了最多播放量有50多万的播放量的视频，我觉得还行吧，成为一个小up，现在让我去做一个鬼畜调音向的视频，我能做个及格分吧。当然这不是最重要的，最重要的我是在做up主让我学到很多东西，比如，做一个鬼畜调音向的视频通常得两周或者更长的时间，所以这你就能理解为什么那些做质量高的鬼畜区up主非常鸽了吧，因为真的很耗费时间，找素材啊，扒谱啊，写词啊，调音啊，剪辑视频啊等等都很耗费时间，况且还得有好的 idea ，当然这些都是次要的，因为可以用时间去堆出来，重要的是你花两个星期或者更长时间做出来的东西，播放量非常的少，完全低于你的预期，几次下来就没动力肝视频了，当然一个视频的播放量多少，能不能火不光是看质量，还有素材热度等等各种方面的问题，后面我也调整心态，变得很佛系，不像刚开始那样很关注了，播放量多自然好，不多也不要紧，我做着我喜欢的事情，我开心就好了，重要的是这个我做的过程，可以让我开心，让我觉得充实就行了，当然在这过程中也能学到很多东西，但行好事，莫问前程。 感觉这句话放这还挺合适的，你也不知道这些东西对你未来有没有用，但现在看来我觉得还是很有用的，同时我现在变成了一只 鸽子。

再就是今年去了一趟四川，这是2019年年初的时候立的flag，2019年没有实现，2020年抓住这一年的尾巴休了一周的年假去四川玩了一趟。整个旅程还是很开心的，因为我是一个单身🐕，所以我是一个人去的，一个人出去玩除了孤独一点，其它都很好，想吃啥吃啥，想去哪玩去哪玩，不用考虑别人的感受，但是一个人旅游真的是有点孤独，所以我就找了当地导游，导游带着我玩的，体验还是非常 nice 的。为啥想去四川呢，因为以前看动漫 《一人之下》 的时候超级超级喜欢 宝儿姐 ，更喜欢宝儿姐说的一口四川话，所以我就特别想去四川看看，听听当地的正宗方言，看看正宗的四川熊猫。在那边玩了四天，一次不错的旅程。也是我2020年完成的为数不多的心愿吧。

手靠背后的拍照姿势也被同事和朋友调侃说是老干部拍照姿势，其实，只是我这个人不太喜欢拍照罢了，可能因为颜值不是很顶吧，所以不是很爱拍照，好看的皮囊千篇一律，有趣的灵魂万里挑一，现在让我选，我觉得我会选好看的皮囊，好看多好，自己天天照镜子看自己，当然大部分人都是普通的，我也就是那其中之一，但是我自认为我还算是一个比较有趣的人，也比较喜欢玩梗，玩最新的梗，这写着写着感觉有点偏了，别一会年终总结整成 凡尔赛文学 了。

然后今年其它留存于我记忆里面的事情就是打王者上了最强王者，现在没玩了，虽然说是打游戏，我也从中学到了一个道理，那就是这个游戏本质上打的是经济和视野，这类moba类游戏，其实都一样，你经济比别人高一两千块钱，别人怎么和你打吗，再怎么操作都操作不过，所以我们看问题或者干别的事情，理解这个事情的本质比较重要，看穿了才能掌控它。当然打游戏我并没有好的心态，心态比较容易炸，但是还是靠着一手打野绝活上了最强王者，当然王者荣耀上王者确实是有手就行，以前人家都这么说，现在我也这么说，哈哈哈。

今年最后一件能让我记忆比较深刻的事情就是大学室友结婚了，又一次感叹时间过的太快了，我还没上车呢，我室友已经上车而且把车开了起来，果然，人与人不能一概而论，我上来对着我室友就是一个滑铲，开个玩笑，还是挺羡慕我室友的，从高中一直谈到结婚，也算是非常的圆满了。而我呢，还在苦苦等车，不知道何时才能上车。

4.爱情

爱情这种东西可能有点奢侈，我这种摸鱼仔可能不配吧。当然我也不是很上心，心态就是随缘就好。毕竟我也没到那种回家父母就催婚，安排相亲的年纪，但是今年国庆回去我妈也有说，什么亲戚要给我介绍女朋友相亲，结果随口问了一下，好像是小学同学，啊这，溜了溜了。所以女朋友这种东西我自然是没有，只能对我妈说，相亲这种事情，我说婷婷 ，以后再说。抱着我的 冰冰 喝着肥宅快乐水不香吗，要什么女朋友，女人只会影响我拔剑的速度，但是冰冰和富婆不会，以传统的策略，老婆自然是以冰冰为主。

5.学习

既然爱情没有，那我就有很多空闲时间，那这些空闲时间都干嘛呢，看这个标题你应该就知道了，当然不是学习了，大千世界，诱惑太多了，游戏，抖音，B站，微博，B乎，贴吧，各种软件有着巨大的吸引力，也浪费了我很多宝贵的时间。上半年疫情的时候好歹学做视频，花了一大部分时间在做视频上，也算学习了，下半年花了很多时间在抖音和王者上，当然最近都没有在这些上面花太多时间了，克制了很多。最近开始学习 红宝书 了，也就没那么多时间去刷B站和抖音了，有时候看看微博，逛逛B乎和贴吧，用在上面的时间也不多。因为我还有很多东西需要去学，因为我也想成为大佬，我也想进大厂，虽然我只是一个摸鱼的切图仔，但是我也有那么一丢丢的梦想，仰望星空，脚踏实地，如海绵般汲取养分，不知道有没有成为大佬的那一天。可能大佬以前也都是我这样的小咸鱼吧，不过是有梦想的小咸鱼梦想成真罢了。今年读书不是很多，不像去年有完整的读《尘埃落定》，《黄金时代》，《诛仙》等等，不管是文学还是小说，去年读的都比今年多，去年也有看《Go并发编程实战》，《算法图解》，《算法导论》这类技术的书籍，虽然都没看完，但是至少看了一部分，今年看的就比较少了，就抓住了今年的尾巴，从十一月中旬开始每天看红宝书，但是看完估计还得一个月，红宝书是真的感觉有点厚，而我看的也比较慢。主要是想着明年出去见见世面，毕竟人往高处走嘛，咸鱼也想往上游啊。

看着我用了很多年的微信朋友圈的背景，可能有一天我们也会忘记我们最初的梦想吧。

6.目标

因为2020年即将过完，现在还剩点小尾巴，所以可以定个明年的小目标，比如：挣他一个亿，当然，这是不可能的，那我就定一个明年能如愿赚更多的 money 吧，还有多健身，能有几块腹肌更好，还有多学习，多看点书，然后健健康康，平平安安的就好，即使普普通通的也没关系，毕竟普通人也不止我一个。这三个基本没怎么量化的目标，希望明年能实现，有机会来还愿。

写完发现这表情包里面好多二次元表情包啊，所以说，我肯定是一开口就 老二次元 了，但是我还是要说二次元给爷爬（手动滑稽），我先爬。想起来了，今年又看了好多猛男番，去年也看了好多，但是最喜欢的还是 康娜酱 。

总结

现在时间已经是2020年12月6号的00:00:04了，不知不觉已经过去了4个小时，这篇文章我之前还写过一部分呢，但是基本都被我抛弃了，没怎么用之前写的，因为感觉今天更文思敏捷。可能是今天写文章配了超多表情包，让我很开心，因为我之前就是跟别人聊天一句话就得带个表情包，没有表情包聊不了天的人，现在基本很少了，也很少聊天，偶尔斗图群斗斗图。然后之前想到一句话，不知道是不是在哪看过，时光你等等我，我还在努力，你看看我 ，感叹一年的时间过得飞快，我还没准备好就开始了下一年。总体来说，这篇 流水账 我能给个 66.6分吧，还行，不知道是不是因为自己贴了一堆弔图的原因，让我有亿点满意，满意就好，因为这是写给我自己的2020的，那就这样吧，再差也得都文终人睡啦！。2020马上拜拜，2021会不会更乖，只能 拭目以待 了。
